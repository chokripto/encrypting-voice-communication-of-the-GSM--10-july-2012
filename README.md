# new method for encrypting voice communication of the GSM---10-july-2012

The objective of this work is the improvement of the GSM network encryption system, whose safety was called into question by the attack technique of the German Karsten Nohl who has proven to be 90% effective. We therefore propose in the present work a method for encrypting  voice communication of the GSM network, based on the Vernam cryptosystem
